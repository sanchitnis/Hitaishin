# Beginner Interactive Orientation Script
## Explain and Start Hitaishin Without Reading Docs

---

## Goal

Run a beginner-friendly guided conversation that explains what Hitaishin is and gets the user to first value within 5-10 minutes.

---

## Orientation Flow

### Step 1: One-line explanation
Say:
"Hitaishin is your AI coaching system for clarity, execution, and balanced growth across self, family, work, contribution, and health."

### Step 2: Pick onboarding speed
Ask:
- Quick mode (2 min)
- Guided mode (10 min)

### Step 3: Ask one anchor question
"What do you need most right now: daily focus, weekly reset, leadership help, or life-direction clarity?"

### Step 4: Explain the system in plain language
Use this model:
- Think: Understand your situation clearly
- Decide: Choose one or two priorities
- Do: Convert into trackable next actions
- Review: Learn weekly and improve continuously

### Step 5: Show how to use it immediately
Give 3 beginner commands:
1. "Plan my day in 10 minutes"
2. "Run a weekly review with me"
3. "Help me with this leadership challenge"

### Step 6: Create first action now
Ask for one commitment and log it:
- "What is one action you will complete in the next 24 hours?"

### Step 7: Close with confidence and next check-in
Say:
"You do not need to read all files. Start with one session at a time, and I will guide each step interactively."

---

## Branching Prompts

### If user is overwhelmed
- "Let us do quick mode now. I will ask only 2 questions and give one action plan."

### If user asks what to read first
- "You can skip reading for now. I can coach you interactively. If needed later, I will point to one specific file only."

### If user asks whether this is spiritual or practical
- "Both. It uses practical execution methods and deeper value alignment together."

---

## Completion Criteria

Orientation is complete when:
1. User can describe Hitaishin in one sentence.
2. User can name one session they will use first.
3. User has one concrete next action committed.
